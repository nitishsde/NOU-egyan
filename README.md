## Project Overview

**NOU e-Gyan Portal** is a Learning Management System (LMS) developed for **Nalanda Open University** during my Summer Training at **Softpro India Pvt. Ltd., Lucknow**.

The portal provides a digital platform for students, administrators, and faculty members to manage academic activities, course materials, student records, and online learning resources efficiently.
